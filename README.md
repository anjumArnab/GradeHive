# GradeHive

GradeHive is a simple student management app built with **Flutter**, **Google Apps Script**, and **Google Sheets**. It allows basic **CRUD operations** (Create, Read, Update, Delete) for managing student records, including **ID**, **Name**, and **Result**.

## Tech Stack

* **Flutter** – Frontend UI
* **Google Sheets** – Backend database
* **Google Apps Script** – API for handling requests
